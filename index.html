🔗 Импорт из Google Drive через Apps Script

Укажите URL веб-приложения Google Apps Script, которое возвращает данные из файла «Выгрузка для себеса и BI» по постоянному File ID на Google Drive.

ℹ️ Apps Script должен возвращать JSON в формате rial_data (структура: file_date, months).
Поддерживается CORS: в скрипте добавьте ContentService.createTextOutput(json).setMimeType(ContentService.MimeType.JSON)
и в doGet() установите return output.setHeaders({'Access-Control-Allow-Origin':'*'})
📄 Загрузка JSON-файла данных

Загрузите файл rial_data.json, сгенерированный Python-скриптом обработки выгрузки 1С.

📊 Загрузка исходного XLSX (требует бэкенд)

Загрузка и обработка файла «Выгрузка_для_себеса_и_BI.xlsx» напрямую в браузере не поддерживается из-за объёма данных (~46 000 строк). Используйте Python-скрипт для генерации JSON, либо Apps Script.
